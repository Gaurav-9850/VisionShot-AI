# VisionShot AI

An AI-powered mobile photography assistant that analyzes the camera frame in real time and provides composition, lighting, and framing suggestions before the user captures a photo.

## Features

- Live camera analysis
- Face detection
- Rule of Thirds guidance
- Lighting analysis
- Blur detection
- Horizon detection
- Composition scoring
- AI-powered recommendations

## Tech Stack

- Flutter
- OpenCV
- MediaPipe
- TensorFlow Lite
- Java Spring Boot (optional backend)
